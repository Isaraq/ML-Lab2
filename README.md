# ML Lab2

## Problem description 
The aim of this  Project is to predict the final student grade (G3) using student diffreant features such as study time, failures, etc. This is a regression problem because the target variable is a continuous numerical value.

## DataSet
Student Perfomeance Dataset
The Dataset contains tabular data about students, including demographic, social, and academic features.
The Target variable is G3, which represents the final student grade.
